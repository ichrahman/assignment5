Important FAQ

What is the Document Object Model (DOM)

DOM stands for Documet Object Model. DOM is an API provided by browser that can be manipulated using JavaScript.

How do you select an element from the DOM?

We can select an element from the DOM using JavaScript. Some common methods are getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes and styles using JavaScript by accessing the element through the DOM.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the context of the DOM where a single event listener is attached to a parent element instead of adding individual listeners to multiple child elements. When an event occurs on one of the child elements, it bubbles up to the parent, allowing the parent to handle the event.
